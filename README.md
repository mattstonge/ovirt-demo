# ovirt-demo
A group of playbooks to manage OPEN SOURCE VIRTUALIZATION
supporting a mix of KVM and RHV
By Matt St. Onge 7/17/2018
_________________
Dependencies
_________________
Ansible 2.5 or higher

libvirt

KVM/QEMU

cockpit

cockpit-machines

and your favorite operating system ISO image (for deployments)
